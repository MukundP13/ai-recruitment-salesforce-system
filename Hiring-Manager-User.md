# Create Hiring Manager User

## Description
This activity involves creating a Hiring Manager user in Salesforce. The Hiring Manager reviews candidate applications and approves or rejects job offers during the recruitment process.

## Steps to Create Hiring Manager User

1. Go to **Setup** in Salesforce.
2. Search for **Users** in the Quick Find box.
3. Click **Users**.
4. Click **New User**.
5. Enter user details:
   - First Name
   - Last Name
   - Email
   - Username
6. Select the **Hiring Manager Profile**.
7. Assign a role if necessary.
8. Click **Save**.

## Responsibilities
The Hiring Manager can:
- Review candidate applications
- Approve or reject job offers
- Monitor recruitment progress
