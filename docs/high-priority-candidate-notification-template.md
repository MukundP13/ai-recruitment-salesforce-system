# High Priority Candidate Application Notification Email Template

## Description
This email template notifies the recruiter or hiring manager when a high priority candidate application is submitted.

## Object
Candidate Application

## Subject
High Priority Candidate Application Received

## Email Body
Dear Hiring Manager,

A high priority candidate application has been submitted.

Candidate Name: {!Candidate_Name}
Position Applied: {!Job_Title}
Application Status: High Priority

Please review the candidate application in Salesforce.

Regards  
Recruitment Team
