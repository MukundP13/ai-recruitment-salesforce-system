# Milestone 13 – Dynamic Forms

## Objective

Enable Salesforce Dynamic Forms to allow flexible field placement and better UI customization.

---

## Steps to Enable Dynamic Forms

Open a Candidate Application record.

Click:

Gear Icon → Edit Page

Lightning App Builder opens.

---

## Upgrade to Dynamic Forms

Select the:

Record Detail Component

Click:

Upgrade Now

Choose:

Candidate Application Page Layout

Click:

Finish

---

## Configure Dynamic Sections

Create sections such as:

### Candidate Information
Fields:
- Candidate
- Job Opening

### Application Details
Fields:
- Application Date
- Experience Level

### AI Evaluation
Fields:
- AI Candidate Score

### Offer Approval
Fields:
- Offer Amount
- Approval Status

---

## Activate Page

Click:

Save → Activate

Choose:

Org Default → Desktop → Phone

Click Save.

---

## Result

Dynamic Forms allow administrators to control field placement and visibility rules for better UI flexibility.
