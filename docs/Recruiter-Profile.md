# Recruiter Profile

## Description
The Recruiter profile is created to allow recruiters to manage candidate applications, job postings, and initiate hiring processes in the recruitment system.

## Object Access
The Recruiter profile has access to the following objects:

- Candidate (Contact)
- Candidate Application
- Job Position

Permissions include:
- Read
- Create
- Edit

## Steps to Create Recruiter Profile

1. Go to **Setup** in Salesforce.
2. Search for **Profiles** in the Quick Find box.
3. Click **Profiles**.
4. Click **New Profile**.
5. Clone from **Standard User** profile.
6. Enter profile name **Recruiter**.
7. Configure object permissions for recruitment-related objects.
8. Click **Save**.

## Purpose
This profile ensures recruiters have the required permissions to manage candidate data and recruitment workflows.
