# Hiring Manager Profile

## Description
The Hiring Manager profile is designed for managers responsible for reviewing candidate applications and approving job offers during the recruitment process.

## Object Access
The Hiring Manager profile has access to:

- Candidate (Contact)
- Candidate Application
- Job Position
- Candidate Offer

Permissions include:
- Read
- Approve
- Edit (limited)

## Steps to Create Hiring Manager Profile

1. Go to **Setup** in Salesforce.
2. Search for **Profiles** in the Quick Find box.
3. Click **Profiles**.
4. Click **New Profile**.
5. Clone from **Standard User** profile.
6. Enter profile name **Hiring Manager**.
7. Configure object permissions required for reviewing applications.
8. Click **Save**.

## Purpose
This profile ensures that hiring managers can review applications and approve candidate offers while maintaining controlled system access.
