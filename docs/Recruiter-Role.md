# Recruiter Role

## Description
The Recruiter role is responsible for managing candidate applications, updating candidate information, and supporting the hiring process.

## Role Hierarchy
The Recruiter role is placed below the Hiring Manager role in the hierarchy. This allows hiring managers to view the records created by recruiters.

## Responsibilities
- Manage candidate records
- Create candidate applications
- Submit candidates for approval

## Steps to Create Recruiter Role

1. Go to **Setup** in Salesforce.
2. Search for **Roles** in the Quick Find box.
3. Click **Roles**.
4. Click **Set Up Roles**.
5. Click **Add Role**.
6. Enter role name **Recruiter**.
7. Place the role below the **Hiring Manager** role in the hierarchy.
8. Click **Save**.

## Purpose
This role allows recruiters to manage candidate data while maintaining a structured hierarchy for recruitment management.
