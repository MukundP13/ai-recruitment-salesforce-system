# Milestone 11 – The Lightning App

## Objective

Create a Lightning Application that serves as the central workspace for recruitment operations.

---

## Lightning App Overview

The Lightning App provides a unified interface for recruiters and hiring managers to manage:

- Job Openings
- Candidate Applications
- Candidate Profiles
- Recruitment Reports

---

## Steps to Create Lightning App

Navigate to:

Setup → App Manager

Click:

New Lightning App

---

## App Details

App Name:

AI-Powered Recruitment & Candidate Management System

Description:

Centralized recruitment management workspace built on Salesforce.

---

## Navigation Items

Add the following items:

- Job Openings
- Candidate Applications
- Contacts
- Reports
- Dashboards

---

## Assign Profiles

Assign the app to:

- System Administrator
- Recruiter
- Hiring Manager

Click Save.

---

## Result

The Lightning App now provides a dedicated recruitment workspace for managing candidate applications and hiring workflows.
