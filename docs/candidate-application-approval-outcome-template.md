# Candidate Application Approval Outcome Notification Email Template

## Description
This email template informs the recruiter about the approval or rejection outcome of a candidate application.

## Object
Candidate Application

## Subject
Candidate Application Approval Outcome

## Email Body
Dear Recruiter,

The candidate application approval process has been completed.

Candidate Name: {!Candidate_Name}
Application Status: Approved / Rejected

Please check the record in Salesforce for further details.

Regards  
Recruitment Team
