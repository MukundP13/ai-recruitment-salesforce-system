# Approval Process

## High Salary Candidate Offer Approval

This approval process ensures that job offers with high salary amounts are reviewed and approved by a Hiring Manager before being finalized.

### Object
Job Application / Candidate Offer

### Criteria
Salary Amount > 1000000 (Example condition)

### Approval Steps

1. **Submit for Approval**
   - Recruiter submits the candidate offer for approval.

2. **Step 1: Hiring Manager Approval**
   - Assigned Approver: Hiring Manager
   - Action: Approve or Reject

3. **Final Approval Actions**
   - Update Application Status → Approved
   - Send notification email to recruiter

4. **Rejection Actions**
   - Update Application Status → Rejected
   - Notify recruiter with reason

### Purpose

- Prevents unauthorized high salary offers
- Ensures hiring manager review
- Maintains hiring policy compliance
