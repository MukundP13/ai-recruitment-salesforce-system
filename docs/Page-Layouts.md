# Milestone 12 – Editing Page Layouts

## Objective

Customize page layouts for recruitment objects to improve usability and data visibility.

---

## Candidate Application Page Layout

Navigate to:

Setup → Object Manager → Candidate Application

Click:

Page Layouts

Open:

Candidate Application Layout

---

## Create Layout Sections

Add the following sections:

### Candidate Information
Fields:
- Candidate
- Job Opening

### Application Details
Fields:
- Application Date
- Experience Level
- AI Candidate Score

### Offer Details
Fields:
- Offer Amount
- Application Status

### Approval Information
Fields:
- Approval Status

---

## Save Layout

Click:

Save

---

## Result

The Candidate Application record page now displays structured sections that improve recruiter usability.
