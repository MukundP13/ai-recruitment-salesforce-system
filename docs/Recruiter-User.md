# Create Recruiter User

## Description
This activity involves creating a Recruiter user in Salesforce. The recruiter is responsible for managing candidate applications, reviewing candidate profiles, and initiating approval processes for hiring.

## Steps to Create Recruiter User

1. Go to **Setup** in Salesforce.
2. In the Quick Find box, search for **Users**.
3. Click **Users**.
4. Click **New User**.
5. Fill in the required details:
   - First Name
   - Last Name
   - Email
   - Username
6. Select the **Recruiter Profile**.
7. Set the role if required.
8. Click **Save**.

## Permissions
The Recruiter user can:
- View candidate applications
- Create and manage job postings
- Submit candidate offers for approval
